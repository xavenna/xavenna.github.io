Detailed Description

A class responsible for assigning textures.

A texturemap is responsible for assigning textures to sf::Sprites. It contains a vector of sf::Textures, which can be assigned using encapsulated methods. It loads the textures from images, using a texturemap file (*.tm). This is a plain-text file that is composed of a list of filenames, which are loaded in order. When setting a texture for a Node, the idTh texture is chosen.
